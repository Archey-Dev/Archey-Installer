# Archey Installer

A beginner-friendly Arch Linux installer with optional learning explanations.

## What is this?

Archey Installer is a custom Arch Linux installation framework designed to:

- Simplify the Arch install process
- Offer optional explanations for learning -- Soon
- Allow desktop environment selection (KDE, GNOME, i3, etc.)
- Provide a cleaner experience than raw archinstall

This is NOT a preconfigured distro.
It is still Arch. You are still in control.

---

## Features

- Minimal base install
- Desktop selection
- Toggleable explanations (learn mode) -- Soon
- Custom user creation
- Custom ISO support

---

## Why?

Arch is powerful, but the manual install can be overwhelming.
This project keeps the power but removes unnecessary friction.
